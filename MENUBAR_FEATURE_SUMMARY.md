# 菜单栏显示模式 - 快速指南

## 🎯 核心功能

优化了 **macOS 顶部菜单栏**（状态栏）的显示内容，让你无需打开应用就能快速了解任务状态。

## 📍 在哪里看到？

在 macOS 屏幕顶部右侧，FlowTask 的图标旁边会显示不同的内容：

```
┌─────────────────────────────────────────────────┐
│  🍎  📁  🔍  ✓ 5  🔋  🔊  📶  🕐  👤          │  ← macOS 菜单栏
└─────────────────────────────────────────────────┘
              ↑
         这里显示任务信息
```

## 🎨 5 种显示模式

| 模式 | 显示效果 | 说明 | 适合场景 |
|------|---------|------|---------|
| **任务数量** | `✓ 5` | 显示待完成任务数 | 快速了解工作量 |
| **今日剩余** | `✓ 3` | 显示今日剩余任务 | 专注今日目标 |
| **完成进度** | `✓ 75%` | 显示完成百分比 | 关注整体进度 |
| **问候图标** | `☀️` | 根据时间变化 | 增加趣味性 |
| **简洁模式** | `✓` | 只显示图标 | 极简风格 |

## ⚙️ 如何设置？

1. 打开 FlowTask 主窗口
2. 点击左侧 "设置"
3. 选择 "桌面贴图"
4. 在 **"菜单栏显示"** 区域选择你喜欢的模式
5. 立即生效！

## ✨ 智能特性

- ⚡ **实时更新**：完成任务后数字立即变化
- 🕐 **定时刷新**：每分钟自动更新（用于时间相关模式）
- 💾 **记住选择**：重启应用后保持你的设置
- 👁️ **预览效果**：设置界面右上角实时预览

## 💡 推荐搭配

- **工作日**：使用"今日剩余"模式，专注当天任务
- **项目冲刺**：使用"完成进度"模式，激励自己
- **日常使用**：使用"任务数量"模式，简单直观
- **极简主义**：使用"简洁模式"，保持桌面整洁

## 🎁 额外福利

除了菜单栏，还可以设置：
- **贴图窗口导航栏**的显示模式
- **悬浮球**的显示位置和大小
- **窗口层级**（始终置顶或普通窗口）

全部在 "设置 → 桌面贴图" 中配置！
